## 2、使用说明

### 2.1、初始化

```vue
npm install
```

### 2.2、启动

```vue
npm run dev
```

### 2.3、打包

```bash
npm run build
```



### 2.4、运行地址

[http://localhost:9528](http://localhost:9528)



**初始密码为：admin/123456**
